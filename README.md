# Xylem-Budget-Extractor
A Web and PDF crawling application to extract budget information

## Getting Started

The Jupyter notebook contains all instructions to run the code and is self explainatory.

### Prerequisites

```
 Pandas
 PyPDF2
 Tabula
 googlesearch
```
